# KotlinWeb
by Allround

## Beschreibung
KotlinWeb ist ein Webframework für Kotlin. 
Das Framework bietet eine einfache Möglichkeit, um auf Requests zu reagieren.
Basieren tut es dabei auf Vert.x Web, was zu einer schnellen Performance führt.
